该目录通常包含以下类型的文件：

- 样式表（CSS、SASS等）
- 字体
- 不会从["public/"](https://vitepress.vuejs.org/zh/guide/directory-structure.html#public-目录)目录中提供的图片。

如果您想从服务器上提供资产，请查看["public/"](https://vitepress.vuejs.org/zh/guide/directory-structure.html#public-目录)目录。
